# PandaFactor 新手教程

欢迎来到 PandaFactor 的世界！这是一份专为新手小白准备的教程，我们会用最通俗易懂的语言，带你从零开始学习如何使用这个量化因子库。

## 📚 教程导航

### 🚀 快速入门（推荐从这里开始）

1. **[快速开始](./01-快速开始.md)** - 5 分钟快速体验，了解 PandaFactor 能做什么
2. **[核心概念理解](./02-核心概念理解.md)** - 用通俗语言理解量化因子的核心概念
3. **[环境搭建](./03-环境搭建.md)** - 详细的环境配置指南，手把手教你安装

### 📖 实战教程

4. **[第一个因子](./04-第一个因子.md)** - 手把手创建你的第一个因子
5. **[因子进阶](./05-因子进阶.md)** - 学习编写复杂的因子
6. **[数据分析](./06-数据分析.md)** - 学会分析因子的效果

### 🔧 进阶内容

7. **[常见问题](./07-常见问题.md)** - 遇到问题？来这里找答案
8. **[最佳实践](./08-最佳实践.md)** - 经验总结，帮你少走弯路

## 🎯 学习路径建议

### 如果你是编程小白

1. 先看 [快速开始](./01-快速开始.md)，了解整体概念
2. 跟着 [环境搭建](./03-环境搭建.md) 完成安装
3. 使用公式方式创建 [第一个因子](./04-第一个因子.md)
4. 遇到问题查看 [常见问题](./07-常见问题.md)

### 如果你有编程基础

1. 快速浏览 [核心概念理解](./02-核心概念理解.md)
2. 完成 [环境搭建](./03-环境搭建.md)
3. 使用 Python 方式创建 [第一个因子](./04-第一个因子.md)
4. 深入学习 [因子进阶](./05-因子进阶.md) 和 [数据分析](./06-数据分析.md)

## 💡 使用建议

-   **循序渐进**：不要跳步，按照顺序学习效果最好
-   **动手实践**：每篇教程都有示例代码，一定要自己运行一遍
-   **遇到问题**：先查看 [常见问题](./07-常见问题.md)，如果找不到答案，可以联系技术支持
