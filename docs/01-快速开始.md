# 快速开始 - 5 分钟快速体验

欢迎来到 PandaFactor！这篇教程将用 5 分钟时间，让你快速了解 PandaFactor 是什么，以及它能为你做什么。

## 🤔 什么是 PandaFactor？

想象一下，你是一个投资分析师，每天需要：

-   分析大量股票数据
-   找出哪些股票可能表现更好
-   评估你的投资策略是否有效

**PandaFactor 就像一个智能助手**，帮你：

-   自动处理海量股票数据
-   快速计算各种投资指标（我们称之为"因子"）
-   评估这些因子的有效性

**简单来说**：PandaFactor 是一个量化因子库，帮你用数据科学的方法做投资分析。

## 🎯 我能用它做什么？

### 1. 创建投资因子

**因子是什么？** 就像给股票打分，分数高的股票可能表现更好。

例如：

-   **动量因子**：最近涨得快的股票，可能还会继续涨
-   **价值因子**：价格便宜的股票，可能被低估了
-   **质量因子**：公司经营好的股票，可能更稳定

### 2. 分析因子效果

创建因子后，系统会自动帮你：

-   计算因子在过去的表现
-   分析因子是否真的有效
-   生成各种图表和报告

### 3. 自动化数据管理

系统会自动：

-   从多个数据源获取最新数据
-   清洗和整理数据
-   定期更新数据

## 🚀 最简单的使用示例

让我们用 3 行代码，看看 PandaFactor 能做什么：

```python
import panda_data

# 初始化数据模块（就像打开工具箱）
panda_data.init()

# 获取一个已经计算好的因子数据
factor = panda_data.get_factor_by_name(
    factor_name="VH03cc651",  # 因子名称
    start_date='20240320',     # 开始日期
    end_date='20250325'        # 结束日期
)

# 现在 factor 变量里就有了这个因子的所有数据！
# 你可以用它做进一步的分析
```

**就这么简单！** 你已经成功获取了因子数据。

## 📊 实际应用场景

### 场景 1：我想找出最近表现好的股票

```python
import panda_data

panda_data.init()

# 获取动量因子（衡量股票最近的表现）
momentum = panda_data.get_factor_by_name(
    factor_name="momentum_factor",
    start_date='20240101',
    end_date='20241231'
)

# 找出因子值最高的股票（最近表现最好的）
top_stocks = momentum.nlargest(10, 'value')
print("最近表现最好的10只股票：")
print(top_stocks)
```

### 场景 2：我想分析一个投资策略

```python
import panda_data

panda_data.init()

# 获取你创建的因子
my_factor = panda_data.get_factor_by_name(
    factor_name="my_custom_factor",
    start_date='20240101',
    end_date='20241231'
)

# 系统会自动帮你分析这个因子的效果
# 包括：收益率、IC值、IR值等指标
```
